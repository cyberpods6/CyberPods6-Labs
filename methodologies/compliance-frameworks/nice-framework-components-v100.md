# nice-framework-components-v100
The article discusses the recent update to the NICE Framework, a nationally focused resource for developing the cybersecurity workforce. In March 2024, the NICE Framework Components v1.0.0 was released, introducing changes to Work Role Categories and descriptions, 11 Competency Areas, a new Insider Threat Analysis Work Role, and updates to Task, Knowledge, and Skill (TKS) statements[1].

## NICE Framework
The NICE Framework establishes a common lexicon for describing cybersecurity work and workers across public, private, and academic sectors. It consists of several key components:

1. Seven Work Role Categories
2. 52 Work Roles
3. Over 2,200 TKS Statements
4. 11 Competency Areas

These components help to standardize and organize cybersecurity work and skills, making it easier for employers to develop their cybersecurity workforce[1].
