## Module 11: Dark Web Forensics

### Understand the Dark Web and Dark Web Forensics
The Dark Web requires specialized routing software to access hidden networks and illegal marketplaces. Forensics focuses on recovering decryption keys, cryptocurrency ledgers, and anonymous browsing traces.

### Determine How to Identify Traces of Tor Browser
Tor browsing activity leaves distinct local artifacts. Investigators use **Windows Prefetch files** to record application execution and specialized memory artifacts.

### Perform Tor Browser Forensics
Using memory analysis to scrape active RAM for unencrypted Tor node connections, browser history, and hidden service URLs before the data is destroyed upon shutdown.

**Tools Used in Module 11:** Tor Browser, WinPrefetchView, Bulk Extractor.
