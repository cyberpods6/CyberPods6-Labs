# Linux Levels of Expertise
Disclaimer: Some of these references may have changed since the creation of this section. They are merely suggestions for more information to on the different levels.


## User Level
- **Command line:** navigation
- **Edit:** files and scripts
- **Manage:** user permissions, run scripts

**References**
- The Linux command line for beginners. (2023). Ubuntu. https://ubuntu.com/tutorials/command-line-for-beginners#1-overview
- Watkins, D. (2022, May 24). 12 essential Linux commands for beginners. Opensource.com. https://opensource.com/article/22/5/essential-linux-commands
- Hess, K. (2019, August 20). An introduction to the VI editor. Enable Sysadmin. https://www.redhat.com/sysadmin/introduction-vi-editor

## Administrator Level
- **Command line:** install and configure software packages
- **Edit:** application configuration files
- **Manage:** applications, services, user and group permissions, capacity

**References**
- Yoshino, B. (2001). Mastering the VI editor. University of Hawai'i College of Engineering. https://alapono.eng.hawaii.edu//Tutor/vi.html
- Carrigan, T. (2023, January 12). . Enable Sysadmin. https://www.redhat.com/sysadmin/3-text-editors-compared
- Linode. (2023, November 29). Linux system administration basics. Linode Guides & Tutorials. https://www.linode.com/docs/guides/linux-system-administration-basics/
- Beginner's guide to Linux system administration. (2021, August 23). GeeksforGeeks. https://www.geeksforgeeks.org/beginners-guide-to-linux-system-administration/

## Networking Level
- **Command line:** display information; application status, IP address information, routing information
- **Edit:** application configurations, network interface information, routing information
- **Manage:** analyze logs, troubleshooting, applications

**References**
- Kenlon, S. (2021, August 2). Learn the networking basics every sysadmin needs to know. Enable Sysadmin. https://www.redhat.com/sysadmin/sysadmin-essentials-networking-basics
- Linux Network Administrators Guide. (n.d.). The Linux Documentation Project. https://tldp.org/LDP/nag2/nag2.pdf
- Technologies, M. (2021, April 22). Linux Networking Commands with Examples. Mindmajix. https://mindmajix.com/linux-networking-commands-best-examples

## Security Level
- **Command line:** update software packages, capture network traffic
- **Edit:** edit bash files, analyze logs
- **Manage:** services utilization, analyze logs, analyze network traffic
