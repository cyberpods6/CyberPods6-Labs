# **Linux CLI Commands**
Here are some resources for common CLI commands that Linux System Administrators would use daily.

## Linux and Windows CLI Commands
Linux commands with their equivalent Windows command line commands:
## 1) Lists files and directories in the current directory
### - (Linux) ls
### - (Windows) dir
## 2) Changes the current directory
### - (Linux) cd
### - {Windows) cd
## 3) Prints the current working directory
### - (Linux) pwd
### - (Windows) chdir
## 4) Copies files or directories
### - (Linux) cp
### - (Windows) copy
## 5) Moves or renames files or directories
### - (Linux) mv
### - (Windows) move or ren
## 6) Removes (deletes) files or directories
### - (Linux) rm
### - (Windows) del or rmdir
## 7) Creates a new directory
### - (Linux) mkdir
### - (Windows) mkdir
## 8) Changes file permissions
### - (Linux) chmod
### - (Windows) icacls
## 9) Displays currently running processes
### - (Linux) ps
### - (Windows) tasklist
## 10) Shows real-time system information, including running processes
### - (Linux) top
### - (Windows) taskmgr
## 11) Shuts down or reboots the system
### - (Linux) shutdown
### - (Windows) shutdown
## 12) Reboots the system
### - (Linux) reboot
### - (Windows) shutdown /r
## 13) Displays network configuration
### - (Linux) ip a or ifconfig or nmcli device show <interface name>
### - (Windows) ipconfig /all

## 14) Shows disk space usage
### - (Linux) df
### - (Windows) wmic logicaldisk get size,freespace,caption
## 15) Displays disk usage of files and directories
### - (Linux) du
### - (Windows) dir /s
## 16) Archives files
### - (Linux) tar
### - (Windows) tar (via Windows Subsystem for Linux) or zip or 7zip
## 17) Searches for patterns in files
### - (Linux) grep
### - (Windows) findstr
## 18) Searches for files in a directory hierarchy
### - (Linux) find
### - (Windows) find
## 19) Securely copies files between hosts
### - (Linux) scp
### - (Windows) pscp (PuTTY)
## 20) Synchronizes files and directories
### - (Linux) rsync
### - (Windows) robocopy
